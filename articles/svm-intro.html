    # 损失函数

    为了将绿色方块的点和红色圆圈的点分开，我们需要找到超平面（在二维空间中是线，三维是平面）。在上图中，直觉告诉我们，B1B_1B1​的线更加好，因为它对训练样本局部扰动的“容忍”性最好。

    # 拉格朗日乘子法

    其实这就是一个凸二次优化问题，有现成的库可以直接求解，但是我们还有更优雅的数学上的解法。

    ”拉格朗日乘子法是一种经典的求解条件极值的解析方法，可将所有约束的优化模型问题转化为无约束极值问题的求解。“我们使用拉格朗日乘子法将上述问题转化为它的”对偶问题“，便于解决。

    令

    θ(w)=max⁡αi≥0L(w,b,α) \theta(w) = \max_{\alpha_i\geq 0}\mathcal{L}(w,b,\alpha) θ(w)=αi​≥0max​L(w,b,α)

    当所有的yi(wTx+b)≥1y_i(\textbf{w}^T\textbf{x} + b) \ge 1yi​(wTx+b)≥1时，θ(w){\theta}(w)θ(w)显然取到最大值12∣∣w∣∣2\frac{1}{2}{||\textbf{w}||}^221​∣∣w∣∣2，否则θ(w)\theta(w)θ(w)能够取到∞\infty∞。所以当满足所有约束条件时，最小化θ(w)\theta(w)θ(w)就是最小化12∣∣w∣∣2\frac{1}{2}{||\textbf{w}||}^221​∣∣w∣∣2。

    下面我们先对w,b​w, b​w,b​求最小，再对α​\alpha​α​求最大。

    为了求出α\alphaα，我们会使用SMO算法。

    针对∑i=1mαiyi=0\sum \limits_{i=1}^m\alpha_iy_i = 0i=1∑m​αi​yi​=0此式，我们假设α1和α2\alpha_1和\alpha_2α1​和α2​是变量，其他的α\alphaα是常量。那么

    α2=1y2(∑i=3nαiyi−α1y1)≜y2(K−α1y1) \alpha_2 = \frac{1}{y_2}\left(\sum_{i=3}^n\alpha_iy_i-\alpha_1y_1\right) \triangleq y_2\left(K-\alpha_1y_1\right) α2​=y2​1​(i=3∑n​αi​yi​−α1​y1​)≜y2​(K−α1​y1​)

    将α2\alpha_2α2​带回原目标函数中，就可以消去α2\alpha_2α2​，那么整个目标函数就是一个关于α1\alpha_1α1​的一元二次函数。同时因为α1和α2\alpha_1和\alpha_2α1​和α2​都有范围，所以可以得到α1\alpha_1α1​的一个取值范围，在这个范围内，我们可以很快的对二次函数求最大值，即完成一次迭代。

    # 软间隔

    我们之前讨论的默认条件是数据是线性可分的。我们在样本空间中找到一个超平面将其完全分割开来，这个超平面是最大化margin来确定的。但是很多情况下，数据不是线性可分的，会有一些outlier，如果我们将这些outlier也算进去，那我们获得的超平面会有很大程度上的过拟合。

    我们原来对所有的样本都要求

    yi(wTxi+b)≥1 y_i(w^Tx_i+b)\geq 1 yi​(wTxi​+b)≥1

    现在我们允许一些样本不满足上述约束，当然，这样的样本应该尽量的少。

    我们改写原来的优化目标：

    min⁡w,b12∣∣w∣∣2+C∑i=1ml0/1(yi(wTxi+b)−1),C>0 \min \limits_{w, b}\frac{1}{2}{||\textbf{w}||}^2 + C\sum \limits_{i=1}^m l_{0/1}(y_i(w^Tx_i+b) - 1), C > 0 w,bmin​21​∣∣w∣∣2+Ci=1∑m​l0/1​(yi​(wTxi​+b)−1),C>0

    这里的CCC用来控制第一项寻找margin最大的超平面和第二项保证数据量偏差最小之间的权重。上式中的$$l_{01}$$ 是一个01损失函数，我们一般使用hinge loss来替换它：

    lhinge(z)=max(0,1−z) l_{hinge}(z) = max(0, 1-z) lhinge​(z)=max(0,1−z)

    原本的分类平面是红色的线，对应的分别是蓝色和粉色的线。但是考虑到黑圈圈起来的蓝点，原来的超平面就偏移到黑色虚线那里了。

    现在我们引入松弛变量ξi≥0\xi_i\geq 0ξi​≥0 ，对应上图黑色线段，据此将约束条件改为

    yi(wTxi+b)≥1−ξi y_i(w^Tx_i+b)\geq 1 - \xi_i yi​(wTxi​+b)≥1−ξi​

    再重写上式

    min⁡w,b12∣∣w∣∣2+C∑i=1mξi \min \limits_{w, b}\frac{1}{2}{||\textbf{w}||}^2 + C\sum \limits_{i=1}^m \xi_i w,bmin​21​∣∣w∣∣2+Ci=1∑m​ξi​

    接下来同上面的拉格朗日乘子法，我们可以解得

    max⁡α∑i=1mαi–12∑i,j=1mαiαjyiyj⟨xi,xj⟩s.t.,0≤αi≤C,i=1,…,m∑i=1mαiyi=0 \begin{align} \max_\alpha &\sum_{i=1}^m\alpha_i – \frac{1}{2}\sum_{i,j=1}^m\alpha_i\alpha_jy_iy_j\langle x_i,x_j\rangle \\ s.t., &0\leq \alpha_i\leq C, i=1,\ldots,m \\ &\sum_{i=1}^m\alpha_iy_i = 0 \end{align} αmax​s.t.,​i=1∑m​αi​–21​i,j=1∑m​αi​αj​yi​yj​⟨xi​,xj​⟩0≤αi​≤C,i=1,…,mi=1∑m​αi​yi​=0​​
